# Building the GUI binary

## GNU/linux

### Install pyPDF

```sh
pip install pypdf2
```
### Install PyInstaller

```sh
pip install pyinstaller
```

### Run the build script

```sh
./create.sh
```

## MacOS

These steps were tested on MacOS Sierra (not High Sierra ).

### Install pyPDF

```sh
sudo easy_install pypdf2
```

### Install PyInstaller

```sh
sudo easy_install pyinstaller
```

### Run the build script

```
./create.sh macos
```
